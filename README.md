# 🐛 Worms

A game made with html5, css3 and JS.

## install

```Shell
git clone
```
## Start
You can use the [Simple-Webview](https://github.com/MarioMatsui/SimpleOverlayWebview), and put the local host in the `./config.js` file.
Or, just start the `./worm.html`.
